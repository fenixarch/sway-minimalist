# sway-minimalist
This is minimalist sway config

Install:
$ sudo pacman -S waybar rofi swaybg alacritty

$ sudo chmod +x ~/.config/sway/launch_spotify.sh

$ sudo chmod +x ~/.config/sway/open_youtube.sh
